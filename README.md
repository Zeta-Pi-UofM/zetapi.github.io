# Zeta Pi Website

This is the Zeta Pi Official Website!

This website is written in React-TypeScript, and is deployed using Vite.

The live version of the site is deployed directly from `main`. Any and all changes should first be done in their respective branch, then merged to `main`.

An archived version of the historical site is available in the `legacy` branch.
